# Oficinaz

Sistema ERP para oficinas mecânicas e auto centers, com foco em produtividade, controle operacional e visibilidade do negócio em um único lugar.

---

## 🌐 Página publicada

Acesse a versão online do projeto:
**https://oficinaz.replit.app/**

---

## 📋 Visão geral

A Oficinaz foi criada para ajudar oficinas e auto centers a reduzirem processos manuais, organizarem a operação e ganharem mais eficiência no dia a dia.

A proposta da solução é centralizar as principais rotinas da oficina — ordens de serviço, controle operacional, acompanhamento financeiro, gestão de equipe e organização da rotina — em um único ambiente.

---

## ✅ Principais benefícios

- Criação de ordens de serviço com mais agilidade
- Maior visibilidade financeira da operação
- Redução de controles manuais e retrabalho
- Centralização de informações em um só lugar
- Mais clareza para a gestão do negócio

## 🚀 Desenvolvimento local

Siga os passos abaixo para rodar o projeto localmente.

### 1. Clonar o repositório
```bash
git clone https://github.com/marceloESPM/Oficinaz.git
```

### 2. Acessar a pasta do projeto
```bash
cd Oficinaz
```

### 3. Instalar as dependências
```bash
npm install
```

### 4. Criar o arquivo `.env.local`

Crie um arquivo chamado `.env.local` na raiz do projeto e configure as variáveis de ambiente:
```env
VITE_REPLIT_APP_ID=your_app_id
VITE_REPLIT_APP_BASE_URL=your_backend_url
```

### 5. Rodar o projeto
```bash
npm run dev
```

Após iniciar, abra no navegador o endereço exibido no terminal — normalmente:

http://localhost:5173
